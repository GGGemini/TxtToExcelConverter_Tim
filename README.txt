Данная программа была создана для моего друга, разрабатывающего печатные платы

Он пользуется программой, которая формирует по его данным отчёт в виде txt-файла,
но его начальство требует с него отчёт в виде excel

Чтобы немного упростить его жизнь, я предложил написать программу-конвертатор

Для этого мы условились о формате данных, в которых он будет вносить значения колонки Value

Заведён небольшой словарик соответствий, чтобы можно было понимать что писать в 1-ую колонку

Также, элементы отсортированы по названиям первой колонки и сгруппированы

Группы разделены объединёнными ячейками
